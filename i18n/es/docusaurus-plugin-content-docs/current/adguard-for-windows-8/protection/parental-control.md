:::info

Este artículo describe AdGuard para Windows v8.0, un bloqueador de anuncios integral que protege su dispositivo a nivel del sistema. Esta es una versión beta que aún está en desarrollo. Para probarlo, descarga la [versión beta de AdGuard para Windows](https://agrd.io/windows_beta).

:::

Hay varios sitios web a los que los niños no deberían poder acceder. _El control parental_ te ayuda a resolver este problema.

![Parental control](https://cdn.adtidy.org/content/kb/ad_blocker/windows/version_8/protection/parental.png)

Cuando _Control parental_ está habilitado, AdGuard analiza el contenido de las páginas web y bloquea aquellas que pueden ser perjudiciales o inapropiadas para los niños. Esto incluye imágenes y texto para adultos, lenguaje vulgar, violencia y contenido relacionado con drogas.

También puedes activar _Búsqueda segura_ para eliminar enlaces inapropiados de los resultados de búsqueda.

Para evitar que los niños descarguen programas no deseados, habilita la opción _Bloquear la descarga de archivos exe_. Esta opción les impide descargar e instalar software.

Puedes gestionar la _lista de bloqueo_ y la _lista de permitidos_ para decidir qué sitios web deben bloquearse y cuáles deben permitirse.

Por último, puedes proteger estos ajustes con una contraseña y elegir a qué cuentas de usuario se aplican.
