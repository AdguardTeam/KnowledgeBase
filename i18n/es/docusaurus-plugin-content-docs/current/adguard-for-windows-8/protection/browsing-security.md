:::info

Este artículo describe AdGuard para Windows v8.0, un bloqueador de anuncios integral que protege su dispositivo a nivel del sistema. Esta es una versión beta que aún está en desarrollo. Para probarlo, descarga la [versión beta de AdGuard para Windows](https://agrd.io/windows_beta).

:::

_La seguridad de navegación_ ofrece una sólida protección contra sitios web dañinos y intentos de phishing.

Aunque AdGuard para Windows no es un antivirus, sí proporciona seguridad esencial al advertirte antes de visitar sitios no confiables o descargar archivos potencialmente peligrosos.

No detendrá un virus que ya se está descargando, ni eliminará malware existente. Pero bloqueará el acceso a sitios web que están marcados como inseguros, ayudándote a evitar amenazas desde el principio.

Para obtener más detalles sobre cómo funciona esta protección, consulte nuestro artículo [Protección contra phishing y malware](/general/browsing-security).

![Seguridad de navegación](https://cdn.adtidy.org/content/kb/ad_blocker/windows/version_8/protection/browsing.png)

En la sección _Seguridad de navegación_, puedes:

- activar notificaciones sobre sitios web maliciosos y de phishing bloqueados;
- activar alertas sonoras;
- permitir enviar a AdGuard datos anónimos relacionados con la seguridad, lo que nos ayuda a mantener actualizada nuestra base de datos de sitios web dañinos.
