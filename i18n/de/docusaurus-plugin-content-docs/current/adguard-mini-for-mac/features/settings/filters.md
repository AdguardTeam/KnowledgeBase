Unter _Einstellungen_ → _Filter_ können Sie einzelne Filter oder ganze Filterkategorien aktivieren.

Filter sind das Rückgrat der Werbeblockierung und im Grunde das, was das Internet sauber hält. Ein Filter ist eine Liste von Regeln, die in einer speziellen Syntax geschrieben sind. Nach diesen Regeln können Inhaltsblocker den Webverkehr filtern und Werbung oder Anfragen an bösartige Websites blockieren.

Die Filter sind in acht thematischen Kategorien zusammengefasst: _Werbeblockierung, Datenschutz, soziale Widgets, Belästigungen, Sicherheit, sprachspezifische Filter, benutzerdefinierte_ und _sonstige Filter_.

Erfahren Sie mehr über [AdGuard-Filter](/general/ad-filtering/adguard-filters) oder [Werbefilterung im Allgemeinen](/general/ad-filtering/how-ad-blocking-works).

Änderungen an den _Filtern_ werden auch unter _Einstellungen_ → _Safari-Erweiterungen_ angezeigt.

## Benutzerdefinierte Filter

![Benutzerdefinierten Filter erstellen](https://cdn.adtidy.org/content/kb/ad_blocker/mini_mac/create-custom.png)

Um einen benutzerdefinierten Filter hinzuzufügen, gehen Sie zu _Einstellungen_ → _Filter_ → _Benutzerdefinierte Filter_ und geben Sie eine URL oder einen lokalen Dateipfad in das entsprechende Feld ein.

Sie können viele Filter auf [filterlists.com](https://filterlists.com/) finden.
