Este artículo describe AdGuard para Windows v8.0, un bloqueador de anuncios integral que protege su dispositivo a nivel del sistema. Esta es una versión beta que aún está en desarrollo. Para probarlo, descarga la [versión beta de AdGuard para Windows](https://agrd.io/windows_beta).

AdGuard puede ampliar la funcionalidad del sitio web actuando como un [director de scripts de usuario](/general/extensions.md). Esto te permite añadir tus propios scripts así como gestionar los existentes.

## Detectar automáticamente userscripts

Cuando esta opción está activada, AdGuard detecta automáticamente cuando tu navegador solicita un userscript y se ofrece a instalarlo. Si lo prefieres, también puedes descargar e instalar scripts de usuario manualmente.

## Scripts de usuario preinstalados

AdGuard viene con cuatro scripts de usuario preinstalados:

- AdGuard Assistant (Beta)
- AdGuard Extra (Beta)
- Bloqueador de Popups AdGuard (Beta)
- Web of Trust (Beta)

### AdGuard Assistant (Beta)

AdGuard Assistant es la versión heredada de Asistente, que puedes usar si el [nuevo Asistente](/adguard-for-windows/browser-assistant.md) no está disponible para tu navegador. Como su nombre indica, te ayuda a gestionar los ajustes básicos de AdGuard directamente desde tu navegador sin abrir la app.

AdGuard Assistant funciona con todos los navegadores compatibles con Microsoft Windows. No aparecerá en sitios web excluidos del filtro ni en ciertas páginas donde pueda causar problemas de visualización.

### AdGuard Extra

AdGuard Extra es una extensión personalizada creada para combatir la eludencia de bloqueadores de anuncios y los métodos de reinyección de anuncios utilizados por sitios web. Estas técnicas se están volviendo cada vez más comunes a medida que los anunciantes intentan saltarse los bloqueadores de anuncios.

AdGuard Extra funciona modificando la forma en que se cargan las páginas, dificultando que los sitios web puedan eludir los bloqueadores de anuncios utilizando estos métodos avanzados.

### Bloqueador de ventanas emergentes AdGuard (Beta)

Esta extensión impide que se abran ventanas emergentes no deseadas mientras navegas. Algunas ventanas emergentes pueden ser útiles — por ejemplo, configuraciones para la gestión de sitios web o información de referencia adicional que aparece tras hacer clic en un enlace. Si es necesario, puedes desactivar el bloqueador de ventanas emergente.

Esta extensión no funciona en los sitios web permitidos.

### Web of Trust (Beta)

Web of Trust te permite ver la reputación de los sitios web basada en las valoraciones de los usuarios. Los sitios web se revisan según varios criterios, incluyendo la confianza y la seguridad.

Esta extensión está desactivada de forma predeterminada. Ten en cuenta que lo desarrolla un tercero, no AdGuard.
