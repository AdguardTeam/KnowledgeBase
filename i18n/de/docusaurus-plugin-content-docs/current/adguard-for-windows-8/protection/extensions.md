:::info

Dieser Artikel beschreibt AdGuard für Windows v8.0, einen umfassenden Werbeblocker, der Ihr Gerät auf Systemebene schützt. Es handelt sich hierbei um eine Beta-Version, die sich noch in der Entwicklung befindet. Um es auszuprobieren, laden Sie die [Beta-Version von AdGuard für Windows](https://agrd.io/windows_beta) herunter.

:::

AdGuard kann die Funktionalität von Websites erweitern, indem es als [Benutzerskript-Verwaltung](/general/extensions.md) fungiert. Dadurch ist es möglich, eigene Skripte hinzuzufügen und bereits vorhandene zu verwalten.

![Erweiterungen](https://cdn.adtidy.org/content/kb/ad_blocker/windows/version_8/protection/extensions.png)

## Benutzerskripte automatisch erkennen

Wenn diese Option aktiviert ist, erkennt AdGuard automatisch, wenn Ihr Browser ein Benutzerskript anfordert, und bietet an, es zu installieren. Wenn Sie möchten, können Benutzerskripte auch manuell heruntergeladen und installiert werden.

## Vorinstallierte Benutzerskripte

AdGuard wird mit vier vorinstallierten Benutzerskripten ausgeliefert:

- AdGuard Assistent (Beta)
- AdGuard Extra (Beta)
- AdGuard Pop-up-Blocker (Beta)
- Web of Trust (Beta)

### AdGuard Assistent (Beta)

AdGuard-Assistent ist die Vorgängerversion von Assistant, die Sie verwenden können, wenn der [neue Assistent](/adguard-for-windows/browser-assistant.md) für Ihren Browser nicht verfügbar ist. Wie der Name schon sagt, können Sie damit grundlegende AdGuard-Einstellungen direkt über Ihren Browser verwalten, ohne die App öffnen zu müssen.

AdGuard-Assistent funktioniert mit allen Browsern, die mit Microsoft Windows kompatibel sind. Sie erscheint nicht auf Websites, die von der Filterung ausgeschlossen sind, oder auf bestimmten Seiten, auf denen sie Darstellungsprobleme verursachen könnte.

### AdGuard Extra

AdGuard Extra ist eine benutzerdefinierte Erweiterung, die entwickelt wurde, um die Umgehung von Werbeblockern und die von Websites verwendeten Methoden zur erneuten Einblendung von Werbung zu bekämpfen. Derartige Techniken werden immer häufiger eingesetzt, da Werbetreibende versuchen, Werbeblocker zu umgehen.

AdGuard Extra modifiziert die Art und Weise, wie Seiten geladen werden, und erschwert es Websites so, Werbeblocker mithilfe dieser hochentwickelten Methoden zu umgehen.

### AdGuard Pop-up-Blocker (Beta)

Diese Erweiterung verhindert, dass beim Surfen unerwünschte Pop-up-Fenster geöffnet werden. Manche Pop-ups können nützlich sein — zum Beispiel Einstellungen für die Website-Verwaltung oder zusätzliche Referenzinformationen, die nach dem Anklicken eines Links erscheinen. Falls erforderlich, können Sie den Pop-up-Blocker deaktivieren.

Diese Erweiterung funktioniert nicht auf Websites, die in der Liste der zugelassenen Websites enthalten sind.

### Web of Trust (Beta)

Mit Web of Trust können Sie die Vertrauenswürdigkeit von Websites anhand von Nutzerbewertungen einsehen. Websites werden anhand verschiedener Kriterien bewertet, darunter Vertrauen und Sicherheit.

Diese Erweiterung ist standardmäßig deaktiviert. Beachten Sie, dass es von einem Drittanbieter und nicht von AdGuard entwickelt wurde.
