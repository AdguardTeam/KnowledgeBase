:::info

Cet article concerne AdGuard pour Android, un bloqueur de publicité multifonctionnel qui protège votre appareil au niveau système. Pour voir comment ça fonctionne, [téléchargez l'application AdGuard](https://agrd.io/download-kb-adblock)

:::

AdGuard pour Android propose une version gratuite et une version payante. Les fonctionnalités payantes renforcent les capacités d'AdGuard :

- _Le blocage des publicités dans les apps_ vous permet de bloquer les annonces dans les applications hors du navigateur. Vous pouvez spécifier des applications à filtrer dans la [_Gestion des applications_](/adguard-for-android/features/app-management)

:::note

AdGuard utilise son propre lecteur multimédia sans publicité pour bloquer les annonces dans les vidéos YouTube. Pour ouvrir le lecteur multimédia, ouvrez l'application YouTube et partagez une vidéo avec AdGuard. Cette fonctionnalité est gratuite.

:::

- _Protection contre le suivi_ augmente votre confidentialité en bloquant les requêtes de suivi, les compteurs en ligne, les balises UTM, les systèmes d'analyse, et plus encore. [En savoir plus sur la Protection contre le suivi](/adguard-for-android/features/protection/tracking-protection)

- _La Navigation sécurisée_ vous avertit si vous êtes sur le point de visiter un site Web potentiellement dangereux. [En savoir plus sur la Navigation sécurisée](/adguard-for-android/features/protection/browsing-security)

- Les _Filtres personnalisés et les règles utilisateur_ vous permettent d'ajouter vos propres règles de filtrage et des filtres tiers pour peaufiner le blocage des publicités. [En savoir plus sur les filtres](/adguard-for-android/features/settings#filters)

- Les _Scripts utilisateur_ vous permettent d'augmenter la fonctionnalité du navigateur et d'utiliser [AdGuard Extra](/adguard-for-android/features/settings#adguard-extra) qui empêche la réinjection de publicités. [En savoir plus sur les scripts utilisateur](/adguard-for-android/features/settings#userscripts)
