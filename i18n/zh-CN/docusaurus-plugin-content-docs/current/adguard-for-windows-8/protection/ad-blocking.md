:::info

本文适用于 Windows 版的 AdGuard v8.0，一款全面的广告拦截器，可在系统层面保护您的设备。 这是一个仍在开发中的 Beta 测试版本。 要试用，请下载[适用于 Windows 的 AdGuard 测试版](https://agrd.io/windows_beta)。

:::

「广告拦截」部分负责移除用户访问的网站及设备上安装的应用中的广告。 为拦截横幅、弹窗、跟踪器及其他侵扰元素，AdGuard 依赖过滤器，即使用[特殊语法编写](/general/ad-filtering/create-own-filters)的规则集。 如想进一步了解其工作原理，请阅读我们[关于过滤器的文章](/general/ad-filtering/how-ad-blocking-works)。

![广告拦截](https://cdn.adtidy.org/content/kb/ad_blocker/windows/version_8/protection/ad_blocking.png)

用户可以查看以下信息：

- 开启基本防护；
- 阻止[搜索广告和自我推广](/general/ad-filtering/search-ads)；
- 禁用 Windows 11 开始菜单中的广告；
- 开启自动激活特定语言的过滤器；
- 管理用户规则。

您还可以打开「过滤日志」，查看电脑上浏览器和应用发出的所有请求。 在日志中，可以轻松通过创建自定义规则来拦截或解除拦截任何请求。 在「过滤器编辑器」中，可以创建、添加和导入自己的规则与过滤器。
