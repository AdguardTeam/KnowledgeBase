:::info

В этой статье описывается AdGuard для Windows v8.0 — комплексный блокировщик рекламы, который защищает ваше устройство на системном уровне. Это бета-релиз, который ещё находится в стадии разработки. Чтобы начать им пользоваться, скачайте [бета-версию AdGuard для Windows](https://agrd.io/windows_beta).

:::

![Поддержка](https://cdn.adtidy.org/content/kb/ad_blocker/windows/version_8/settings/support.png)

Этот раздел будет полезен, если у вас есть вопросы или предложения по улучшению AdGuard для Windows v8.0. Рекомендуем ознакомиться с [FAQ](https://adguard.com/support/faq.html) и этой Базой знаний до обращения в поддержку.

Если вы заметили пропущенную рекламу, пожалуйста, сообщите нам через вкладку _Сообщить о некорректной блокировке_.

Вы также можете присоединиться к нашим социальным каналам, чтобы обсудить AdGuard с другими пользователями, или связаться с нами на GitHub, чтобы предложить новые функции или сообщить об ошибках.
