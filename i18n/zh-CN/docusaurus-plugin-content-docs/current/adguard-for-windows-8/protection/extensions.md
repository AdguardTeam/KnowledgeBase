AdGuard 可作为[用户脚本管理器](/general/extensions.md)，用于扩展网站功能。 借助此功能，您可以添加自己的脚本，并管理现有脚本。

![扩展](https://cdn.adtidy.org/content/kb/ad_blocker/windows/version_8/protection/extensions.png)

## 自动检测用户脚本

启用此选项后，当您的浏览器请求用户脚本时，AdGuard 会自动检测并提供安装。 如果您愿意，也可以手动下载并安装用户脚本。

## 预装用户脚本

AdGuard 预装了四个用户脚本：

- AdGuard 助手（Beta）
- AdGuard Extra（Beta）
- AdGuard 弹窗拦截器（Beta）
- Web of Trust（Beta）

### AdGuard 助手（Beta）

AdGuard 助手是旧版助手，当[新版助手](/adguard-for-windows/browser-assistant.md)不适用于您的浏览器时，可使用此版本。 顾名思义，它可以帮助您直接从浏览器中管理 AdGuard 的基本设置，无需打开应用程序。

AdGuard 助手适用于所有与 Microsoft Windows 兼容的浏览器。 它不会出现在被排除过滤的网站上，也不会出现在某些可能导致显示问题的页面上。

### AdGuard Extra

AdGuard Extra 是一个自定义扩展，用于对抗网站所使用的绕过广告拦截器及重新注入广告的方法。 随着广告商试图绕过广告拦截器，这类技术正变得越来越普遍。

AdGuard Extra 通过修改页面加载方式来运作，使网站更难利用这些高级方法绕过广告拦截器。

### AdGuard 弹窗拦截器（Beta）

此扩展可在您浏览时阻止不需要的弹窗打开。 某些弹窗可能是有用的，例如网站管理设置或点击链接后出现的额外参考信息。 如有需要，您可以关闭「弹窗拦截器」。

此扩展在允许列表中的网站上不起作用。

### Web of Trust（Beta）

Web of Trust 让用户根据用户评价查看网站的信誉。 网站会根据信任度、安全性等多个标准进行评估。

此扩展默认关闭。 请注意，它由第三方开发，并非 AdGuard。
