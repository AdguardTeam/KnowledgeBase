## Safari Web拡張機能（Safari web extension）

AdGuard for iOS には、Safari と統合して効率的な広告ブロック、プライバシー保護、カスタマイズ可能なフィルタをブラウザ内で直接提供する Web 拡張機能というものが使われています。
