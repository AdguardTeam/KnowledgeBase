더 많은 필터가 필요한 경우, **AdGuard 커스텀**에 필터를 추가할 수 있습니다. 사용자 정의 필터를 추가하려면 해당 필드에 URL 또는 로컬 파일 경로를 입력하세요. [filterlists.com](https://filterlists.com/)에서 새 필터를 찾을 수 있습니다.
