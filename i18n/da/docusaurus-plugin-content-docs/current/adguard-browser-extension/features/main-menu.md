:::info

Denne artikel omhandler AdGuard Browser Extension, der kun beskytter selve webbrowseren. For at beskytte hele enheden, [download AdGuard-appen](https://agrd.io/download-kb-adblock)

:::

Udvidelsens hovedside kan tilgås ved at klikke på dens ikon på webbrowserens værktøjsbjælke.

![Hovedmenu \*mobile_border](https://cdn.adtidy.org/content/Kb/ad_blocker/browser_extension/main_popup_mv2.png)

Fra hovedsiden kan beskyttelse håndteres og nøglefunktioner tilgås:

- Skjul manuelt ethvert element på en webside (en tilpasset regel føjes automatisk til _Brugerregler_)
- Åbn _Filtreringslog_ for at se detaljerede oplysninger om webbrowsertrafikken og blokere forespørgsler i farten
- Indsend en klage over oversete annoncer eller ødelagte sideelementer, så vores filteringeniører kan gennemgå og løse problemet
- Tjek et websteds sikkerhedsrapport
- Se statistik over alle blokerede elementer

By clicking the icons in the top-right corner of the extension’s main menu, you can manually update filters, pause protection, or open the extension settings.
