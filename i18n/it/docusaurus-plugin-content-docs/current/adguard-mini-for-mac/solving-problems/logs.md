:::info

Questo articolo riguarda AdGuard Mini per Mac, che protegge solo il tuo browser Safari. Per proteggere completamente il tuo Mac, [scarica l'app AdGuard](https://agrd.io/download-kb-adblock)

:::

Per analizzare e diagnosticare vari problemi che possono verificarsi con AdGuard Mini per Mac, il servizio di assistenza AdGuard può aver bisogno dei suoi file di annotazioni. Questi file contengono registrazioni di errori e altri eventi software. Di seguito sono riportate brevi istruzioni su come raccogliere i registri e inviarli al supporto, se necessario.

### Come raccogliere i registri

1. Apri AdGuard Mini.
2. Vai in _Impostazioni_, scorri in basso e clicca _Esporta annotazioni_.
3. Scegli una posizione in cui salvare il file .zip e fai clic su _Salva_.

Ora puoi inviare questo file a [support@adguard.com](mailto:support@adguard.com) o allegarlo alla tua segnalazione di errore su GitHub.

:::note

AdGuard si impegna a proteggere la tua privacy. Seguiamo rigorosamente la nostra [Politica di riservatezza](https://adguard.com/privacy.html) e non raccogliamo alcuna informazione privata sugli utenti. Prima di inviare i registri al team di supporto, esamina il file poiché potrebbe contenere informazioni aggiuntive che non desideri condividere. Se contiene tali informazioni personali, ti consigliamo di eliminarle prima.

:::

### Come allegare gli registri a una segnalazione di bug su GitHub

1. Vai allo [spaccio GitHub di AdGuard Mini per Mac](https://github.com/AdguardTeam/AdGuardMiniForMac/issues) e clicca _Nuovo problema_.
2. Invia il file di registri a [devteam@adguard.com](mailto:devteam@adguard.com). Includi l'ora del bug e allega un link al tuo problema o il suo numero (appare come #numero accanto al titolo).
   In alternativa, puoi caricare il file di registri in Google Drive e inviarlo a [devteam@adguard.com](mailto:devteam@adguard.com). Aggiungi il link del file al tuo problema su GitHub.
