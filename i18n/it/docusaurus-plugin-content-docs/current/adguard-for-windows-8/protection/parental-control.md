:::info

Questo articolo descrive AdGuard per Windows v8.0, un blocca-inserzioni completo che protegge il tuo dispositivo a livello di sistema. Questo è una versione beta ancora in fase di sviluppo. Per provarlo, scarica la [versione beta di AdGuard per Windows](https://agrd.io/windows_beta).

:::

Ci sono diversi siti web ai quali i bambini non dovrebbero poter accedere. _Controllo parentale_ ti aiuta a risolvere questo problema.

![Controllo parentale](https://cdn.adtidy.org/content/kb/ad_blocker/windows/version_8/protection/parental.png)

Quando il _Controllo parentale_ è abilitato, AdGuard scansiona il contenuto delle pagine web e blocca quelle che possono essere dannose o inadeguate per i bambini. Questo include immagini e testi per adulti, linguaggio volgare, violenza e contenuti relativi a droghe.

Puoi anche attivare la _Ricerca sicura_ per rimuovere collegamenti inappropriati dai risultati della ricerca.

Per impedire ai bambini di scaricare programmi indesiderati, abilita l'opzione _Blocca scaricamento file exe_. Questa opzione li blocca dallo scaricare e installare programmi.

Puoi gestire la _Lista blocchi_ e la _Lista consensi_ per decidere quali siti web devono essere bloccati e quali devono essere consentiti.

Infine, puoi proteggere queste impostazioni con una parola d'accesso e scegliere a quali profili utente si applicano.
