Benutzerregeln können verwendet werden, um die Werbeblockierung individuell anzupassen, wenn Sie etwas Persönliches möchten: eine Website vom Filtern ausschließen, Anfragen entsperren oder Skripte auf einer bestimmten Website erlauben. Sie können manuell hinzugefügt oder importiert werden. Sie werden automatisch erstellt, wenn Sie ein Element auf der Seite blockieren.

Um Ihre eigenen Filterregeln hinzuzufügen, verwenden Sie eine [spezielle Syntax](/general/ad-filtering/create-own-filters).

### Regelkonstruktor

Wir wissen, dass es nicht jedermanns Sache ist, Filterregeln selbst zu schreiben. Deshalb können Sie einfach auf _Regel erstellen_ klicken, um einen Regelkonstruktor zu verwenden: Wählen Sie aus, was Sie tun möchten, klicken Sie sich durch ein paar Optionen, und schon sind Sie fertig. Blockieren Sie Bilder, Videos, Skripte oder andere Elemente, die Sie lieber nicht auf einer Seite sehen möchten.

![Regelkonstruktor](https://cdn.adtidy.org/content/kb/ad_blocker/mini_mac/build-a-rule.png)

Hier ein Beispiel für einen kniffligen Fall – ein Cookie-Pop-up, das sich der Filterung widersetzt. Suchen Sie die entsprechende Stelle im Seitenquelltext, öffnen Sie dann die _Benutzerregeln_ und erstellen Sie eine Regel, um dies zu blockieren:
