sidebar_position: 1

:::info

Dieser Artikel behandelt AdGuard für iOS, einem multifunktionalen Werbeblocker, der Ihr Gerät auf Systemebene schützt. Um zu sehen, wie es funktioniert, laden Sie [die AdGuard-App](https://agrd.io/download-kb-adblock)herunter.

:::

Es gibt zwei Optionen, um Premium-Funktionen in der AdGuard für iOS-App zu aktivieren:

1. **Ein Abonnement kaufen.** Tippen Sie einfach irgendwo in der App auf das Schild **Premium holen** und folgen Sie den Anweisungen auf dem Bildschirm. Sie können ein Abonnement abschließen, indem Sie irgendwo in der App oder auf der Registerkarte *Lizenz* auf das *Premium-Label* tippen. Neukunden können eine 7-tägige kostenlose Testphase nutzen.

2. **Eine AdGuard-Lizenz [auf unserer Website](https://adguard.com/license.html) kaufen.** Um sie zu aktivieren, öffnen Sie Ihre AdGuard-App und tippen Sie *Einstellungen → Lizenz → Anmeldung*, und geben Sie dieselbe E-Mail-Adresse ein, die Sie für den Kauf verwendet haben. Sie können auch den Aktivierungscode in das E-Mail-Feld eingeben und das Passwortfeld leer lassen. Wir empfehlen jedoch, die Kombination aus E-Mail und Passwort zu verwenden, da dies eine sicherere Option ist.

:::note

AdGuard Pro für iOS ist unsere andere iOS-App. Sie kann nur im [App Store](https://apps.apple.com/app/adguard-pro-adblock-privacy/id1126386264)gekauft werden.

:::
