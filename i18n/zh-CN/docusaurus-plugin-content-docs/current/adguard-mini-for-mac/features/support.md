「支持」部分包含有用的选项，当您遇到问题或疑问时可以使用：

- 常见问题

  许多常见问题和解答都可以在这里找到

- 报告错误拦截

  跳转到可以详细报告错误或问题的页面。

- 联系支持

  直接从应用中联系我们的支持团队，并允许附加日志和应用信息。

  ![联系支持屏幕](https://cdn.adtidy.org/content/kb/ad_blocker/mini_mac/contact-support.png)

- 讨论

  跳转到包含 AdGuard 社交媒体和 GitHub 链接的页面
